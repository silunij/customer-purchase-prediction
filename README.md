# Customer Purchase Prediction using Logistic Regression
## overview
This project builds a logistic regression model in R to predict whether a customer will make a purchase based on behavioral and demographic features. The goal is to identify key factors influencing purchase decisions and demonstrate how statistical modelling can support data-driven marketing strategies.

## Dataset
The dataset contains customer behavioral and demographic information used to predict purchase decisions. This dataset was collected from an observational study and includes data about browser sessions of users visiting an online shopping website.

The target variable indicates whether the customer completed a purchase.

## Methodology

The project follows a standard machine learning workflow:

1. Data cleaning and preprocessing
2. Exploratory data analysis (EDA)
3. Feature selection using StepAIC
4. Logistic regression model training
5. Model evaluation using statistical metrics
